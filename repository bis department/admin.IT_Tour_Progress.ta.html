Enthält alle aktuellen Fahrten und deren Fortschritt zu einer Zugmaschine.
Column Name Type Default Comment
VehicleId uniqueidentifier Gibt die FahrzeugId an. Ist immer eine Zugmaschine.
StartAddressId uniqueidentifier Gibt die Startadresse der Fahrt an
EndAddressId uniqueidentifier Gibt die Zieladresse der Fahrt an
TourId uniqueidentifier Gibt die Tour-Id an, die aktuell gefahren wird. Ist die Aktuelle Fahrt eine Leerfahrt, wird die nächste Tour angegeben
Distance integer Gibt die Distanz in Kilometer bis zum Zielort an
RequiredTimeInMinutes integer Gibt die Zeit an, die noch zum Ziel benötigt wird. Zwischenziele eingeschlossen
CurrentLocationText varchar(100) Gibt den Ort an, an dem sich das Fahrzeug aktuell befindet
CurrentLongitude decimal(9, 6) Aktuelle Longitude des Fahrzeugs
CurrentLatitude decimal(9, 6) Aktuelle Latitude des Fahrzeugs
IsEmptyTour bigint 0 Gibt an, ob es eine Leerfahrt ist oder nicht.
LastProcessedTelematicFiles timestamp Gibt das Datum der Telematikinformationen an, die zuletzt verarbeitet wurde. Bei einem Durchlauf werden nur solche Daten analysiert, die neuer sind.
EndDate date Gibt das Enddatum einer Tour an
EndTime time Gibt die Endzeit einer Tour an
DurationInMinutes integer 0
StartDate date
StartTime time
StartLatitude decimal(9, 6)
StartLongitude decimal(9, 6)
EndLatitude decimal(9, 6)
EndLongitude decimal(9, 6)
IsEndOfTours bit 0
